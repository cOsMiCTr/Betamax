GET /movies/{title}

Returns the movie details of the movies from the database

Parameters Required Description
apiKey Not Required API Key required for all API calls. Register on site to get free API Key.
title Required Expression for search. For examples "The Dark Knight" or "Inception".
Response Returns a JSON array of the movie objects in the database with the title you searched for:
  • ID: Int
  • Title: String
  • Description: String
  • Director: String
    • Name: String
    • Bio: String
    • Birth Year: Date
    • Death Year: Date
  • Genre: String
  • Release Date: Date
  • Image URL: String
